# TraninSimple PHP Training Project (Login & Registration)

This is a basic PHP training project that demonstrates how to build a simple website with:

Home Page

Registration Page

Login Page

It is designed for beginners to understand the fundamentals of PHP, MySQL, and Bootstrap.

📌 Project Overview

This project helps you learn:

How to create forms in HTML

How to handle form data using PHP

How to connect PHP with MySQL database

Basic login and registration system

📁 Files Included

index.php – Home Page

register.php – Registration Page

login.php – Login Page

config.php – Database Connection

🛠️ Technologies Used

HTML

CSS

Bootstrap

PHP

MySQL

⚙️ Setup

Install XAMPP / WAMP

Start Apache and MySQL

Create a database named user_system

Import/create users table

Place project in htdocs folder

Run project in browser

🎯 Purpose

This project is only for training and learning purposes. It shows a simple implementation of user authentication using PHP.
